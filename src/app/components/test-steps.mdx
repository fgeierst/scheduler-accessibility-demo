# Test Steps

Use this flow to manually compare scheduler and calendar implementations. Run the same task sequence twice:

- Keyboard
- Screenreader

Use Storybook's isolation mode `Option Shift I`.

## User Tasks

- Find out how many events are in selected week
- Navigate to a specific event (Lunch with Llama), find out when it is, and open the event details
- Add a New Event (Chat with Cat at Wed 2nd 9:00am)
- Find out what the current date is and switch to Day View
