## Keyboard Functionality

These shortcuts are documented by Microsoft for **new Outlook for Windows**.

| Shortcut | Action | Status |
| --- | --- | --- |
| <kbd>Ctrl</kbd> + <kbd>2</kbd> | Open Calendar | Documented by Microsoft |
| <kbd>Ctrl</kbd> + <kbd>N</kbd> | Create a new calendar item | Documented by Microsoft |
| <kbd>Enter</kbd> | Open the selected item | Documented by Microsoft |
| <kbd>Delete</kbd> | Delete the selected item | Documented by Microsoft |
| <kbd>Ctrl</kbd> + <kbd>Left Arrow</kbd> / <kbd>Ctrl</kbd> + <kbd>Right Arrow</kbd> | Move to the previous or next day in Day view | Documented by Microsoft |
| <kbd>Alt</kbd> + <kbd>Down Arrow</kbd> / <kbd>Alt</kbd> + <kbd>Up Arrow</kbd> | Move to the previous or next week in Week and Work Week views | Documented by Microsoft |
| <kbd>Alt</kbd> + <kbd>Page Down</kbd> / <kbd>Alt</kbd> + <kbd>Page Up</kbd> | Move to the previous or next month in Month view | Documented by Microsoft |
| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>2</kbd> | Switch to Work Week view | Documented by Microsoft |
| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>3</kbd> | Switch to Full Week view | Documented by Microsoft |
| <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>4</kbd> | Switch to Month view | Documented by Microsoft |
| <kbd>Alt</kbd> + <kbd>1</kbd> through <kbd>Alt</kbd> + <kbd>10</kbd> | Show 1 to 10 days starting from today | Documented by Microsoft |
| <kbd>Ctrl</kbd> + <kbd>Period</kbd> | Browse events in the current view | Documented by Microsoft |
| <kbd>Ctrl</kbd> + <kbd>Comma</kbd> | Move to the previous event | Documented by Microsoft |

Microsoft also documents that shortcuts are based on the US keyboard layout and that new Outlook differs from classic Outlook for Windows.

## Semantic Structure

Microsoft documents the calendar experience in new Outlook for Windows as a screen-reader-friendly UI with identifiable regions rather than raw HTML landmarks.

From the official screen-reader guidance, the calendar main view exposes these navigable regions:

- The `New event` button
- The date picker
- The calendar list
- The view switcher
- The calendar view options
- The calendar view

Microsoft also states that when you open Calendar, screen readers announce the current date, whether there are events for that day, and the total number of events for the current month. When focus is on a button or selectable element, pressing `Enter` performs the action or selects the element.

### Confirmed Microsoft Guidance

| Guidance | Meaning |
| --- | --- |
| `Tab` | Move focus through screen elements |
| `Ctrl` + `F6` | Cycle through main regions |
| `Enter` | Activate the current button or selectable element |
| `Month view` event browsing | Screen readers announce individual events and allow browsing through them |

### Implication For This Repo

For a web scheduler in this repository, semantic review usually means HTML structure and ARIA. For Outlook for Windows, the equivalent review target is the Windows accessibility model and screen-reader navigation order described by Microsoft.

This entry intentionally does not claim local verification, because no Windows Outlook instance is available in this workspace.
